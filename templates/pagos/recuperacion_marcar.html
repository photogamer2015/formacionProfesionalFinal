Cómo usar el descuento con Abono + Módulo v

Este método solo se usa cuando la recuperación se descuenta de un pago ya registrado como Abono + Módulo.

  1. Selecciona el módulo que el estudiante perdió.
  2. Si ese módulo tiene un Abono + Módulo disponible, se habilita la opción de descontar.
  3. Si la opción aparece gris, registra la recuperación normalmente para cobrarla después.
  4. Al descontar, se usa el recibo existente y no se crea un cobro nuevo.
Los abonos normales y los pagos Solo Módulo no habilitan este descuento.
